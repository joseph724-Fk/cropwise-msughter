# CropWise

CropWise is a smart agricultural advisory platform developed by **Msughter Sulega** to help Lagos farmers choose suitable crops based on soil type, season, and location.

## Project Description

CropWise helps smallholder farmers make better farming decisions by recommending crops for the current planting season. The system uses crop data, soil type, water requirements, expected yield, and location-based logic to suggest suitable crops.

## Features

- Public homepage showing best crops to plant in the current month
- Farmer registration and login
- Personalized dashboard based on LGA and soil type
- Crop library and crop detail pages
- Farm tracker for planted crops
- Fertilizer, weeding, and harvest schedule reminders
- Same-LGA farmer community chat
- Smart rule-based CropWise Assistant
- PWA support with manifest, icon, service worker, and offline page
- Offline farm record saving and sync support
